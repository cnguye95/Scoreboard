# Scoreboard

A daily log of verifiable, public actions.

This exists to keep me accountable and to give me a daily scoreboard to
measure progress. The rule: only things that produce an artifact someone
else could verify get logged — a pushed commit, an application sent, an
event attended, a message to my accountability partner. Understanding,
studying, and figuring-things-out do not count, no matter how much work
they felt like. Output only.

## Format

Each day gets a dated entry. One line minimum. The question is binary:
did a real shot go over the net today — yes or no.
